## Description
CorroCheck is a mobile application designed to modernize and simplify the process of corrosion inspection for engineers, inspectors, and maintenance personnel. The app allows users to capture or upload images of metallic structures, classify the type of corrosion present — including uniform, galvanic, pitting, and crevice corrosion — and assign a severity level of Low, Medium, or High. Each inspection is saved as a detailed report containing the image, corrosion type, severity rating, GPS or manually entered location, date, and notes, all of which can be retrieved and reviewed at any time through the app's history feature. Built with React Native and Expo for cross-platform mobile use, and powered by Firebase for secure cloud storage and user authentication, CorroCheck aims to reduce manual errors and improve the consistency and accessibility of corrosion inspection data in the field.
